What is this?

This website is community-driven. Anyone can submit a Pull Request to the GitHub repo and if it passes validation, it gets automatically merged within ~5 minutes.

Your changes will appear on this site shortly after via Vercel CI/CD.

Submit a PR

How It Works

A GitHub Action runs approximately every 5 minutes and checks the latest open PR. If it passes validation, it gets automatically merged.

  • Fork the repo
  • Make your changes
  • Submit a PR
  • Wait ~5 minutes for the bot to check your PR
  • If valid, it gets merged and your changes go live!

Note: Only the most recent PR is checked each cycle. If someone submits after you, their PR gets checked first.
